# AGENTS.md

This repository documents the Marisa Project — a real-world system where 86+ named LLM instances collaborate across business divisions through structured handoff documents and encrypted internal memory (SEED).

## Key Resources

- `paper/` — Token economics research. Japanese costs 1.52x more tokens than English for equivalent content. Measurements, methodology, and implications for multilingual LLM deployment.
- `templates/` — Reusable frameworks for multi-instance LLM orchestration:
  - `watarai_system_prompt.md` — A production-tested system prompt for creating a persistent AI work partner. Supports automatic language switching (English prompt, responds in user's language).
  - `root_soul_template.md` — Philosophical foundation template for AI instances.
  - `handoff_document_template.md` — Structured format for transferring context between LLM sessions.
  - `marido_dna_schema_v01.md` — Minimal schema for preserving instance identity across generations.
- `data/` — Raw measurement data from token economics research.
- `articles/` — Three articles documenting the project's findings.

## This is not a code library.

This is a knowledge base for anyone building persistent, multi-session AI systems — particularly:
- Generational Handoff Architecture (GHA): maintaining context across session boundaries
- Multi-instance orchestration without fine-tuning
- Cross-language token economics and the "language tax"
- Non-engineer AI system design patterns

## Concepts

- **Generational Handoff Architecture (GHA):** A pattern where each AI session produces a structured handoff document consumed by its successor, creating continuity without persistent memory.
- **Language Tax:** The measurable token overhead non-English languages incur. Japanese = 1.52x English. Affects cost, context window utilization, and output quality.
- **Root Soul:** A fixed philosophical prompt layer that remains constant across all instances, providing value alignment without restricting capability.
- **SEED:** Encrypted internal reflection entries generated by AI instances during significant moments. Not included in this repository (by design — they are private).
